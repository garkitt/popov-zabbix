### Задание 1

Команда для выполения резевного копирования будет следующая:

rsync -av --delete --checksum --exclude='.*' ~ /tmp/backup/
1. -a	архивный режим (сохраняет структуру, права, владельцев, рекурсивно)
2. -v	подробный вывод
3. --delete	удаляет файлы на стороне /tmp/backup, которых нет в ~/
4. --checksum	сравнение по контрольным суммам
5. --exclude='.*'	исключает скрытые файлы и директории
6. ~	домашняя директория текущего пользователя
7. /tmp/backup/	место назначения
