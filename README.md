### Задание 1

Команда для выполения резевного копирования будет следующая:

*rsync -av --delete --checksum --exclude='.*' ~ /tmp/backup/
-a	архивный режим (сохраняет структуру, права, владельцев, рекурсивно)
-v	подробный вывод
--delete	удаляет файлы на стороне /tmp/backup, которых нет в ~/
--checksum	сравнение по контрольным суммам
--exclude='.*'	исключает скрытые файлы и директории
~	домашняя директория текущего пользователя
/tmp/backup/	место назначения

![Скрин шаблона](https://github.com/garkitt/popov/blob/4d620da969bc71cbae9ebe4b724f3f3f9b9d079a/img/1_rsync.png)`
